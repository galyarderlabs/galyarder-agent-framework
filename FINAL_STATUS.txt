│ QUICK START                                                                  │

  Standalone Mode:
    1. Install Framework in Claude/Cursor
    2. Chat with galyarder-specialist
    3. Done!

  Dashboard Mode:
    1. cd dashboard
    2. pnpm install
    3. pnpm dev
    4. Open http://localhost:3100
    5. Create company, hire agents!

  Test Integration:
    cd dashboard/packages/adapters/galyarder-framework
    node test.js
